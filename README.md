# 跑团社交平台

全栈跑团社交平台，支持创建和加入跑团、约跑活动、动态发布和跑团挑战赛。

## 功能简介

- **跑团管理**：创建或加入跑团，跑团主页展示成员列表、本周跑量排行和近期活动
- **约跑活动**：团长发起约跑，填写时间、地点、路线和预计配速，成员报名参加，跑完后晒成绩和照片
- **个人主页**：展示累计跑量、最好成绩、跑过的赛事奖牌墙
- **动态社交**：发布跑步感受，支持点赞和评论，查看关注的人的动态流
- **挑战赛**：跑团之间发起月度跑量挑战赛，比拼总跑量
- **管理后台**：跑团审核、内容审核、举报处理、排行榜管理

## 技术栈

- 前端：Vite + React 18 + TypeScript + React Router v6
- 后端：Express + TypeScript + better-sqlite3
- 认证：JWT + bcryptjs

## 运行方式

```bash
npm install
npm run dev
```

前端启动在 http://localhost:5173，后端启动在 http://localhost:3001。

首次启动会自动创建 SQLite 数据库，插入 5 个用户、3 个跑团、4 条动态、奖牌和挑战数据。
